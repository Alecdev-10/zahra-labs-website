# Site Web de ZAHRA LABS

Site vitrine officiel de **ZAHRA LABS**, entreprise béninoise spécialisée dans les technologies de l'information, le conseil, l'innovation et la transformation digitale.

Ce projet a été développé avec **React** afin de proposer une plateforme moderne, responsive et intuitive présentant les activités, les domaines d'expertise et les services de l'entreprise.

---

## À propos de ZAHRA LABS

**ZAHRA LABS** est une strtup béninoise qui accompagne les institutions publiques, les entreprises privées, les organisations internationales, les PME, les startups et les porteurs de projets dans leur transformation numérique grâce à des solutions innovantes, durables et adaptées à leurs besoins.

Ses domaines d'intervention couvrent notamment :

- le conseil et l'ingénierie de projets ;
- la transformation digitale ;
- le développement de solutions numériques ;
- l'intelligence artificielle ;
- la recherche et développement (R&D) ;
- l'économie circulaire et l'AgriTech ;
- les technologies de la santé (HealthTech) ;
- la formation professionnelle.

---

## Fonctionnalités

Le site permet notamment de :

- Présenter l'entreprise et sa vision ;

## Installation

Cloner le dépôt :

```bash
git clone https://github.com/votre-utilisateur/zahra-labs-website.git
```

Se placer dans le dossier du projet :

```bash
cd zahra-labs-website
```

Installer les dépendances :

```bash
npm install
```

Lancer le serveur de développement :

```bash
npm run dev
```

Créer une version de production :

```bash
npm run build
```

Prévisualiser la version de production :

```bash
npm run preview
```
